# Installation
#### Clone the repository
```
git clone https://github.com/Bingo76/sysup.git
```
#### Give permissions to the script
```
chmod +x /sysup/setup.sh
```
#### Run the script with sudo
```
sudo ./setup.sh
```
#### Configure .xinitrc
```
Include in .xinitrc
feg --bg-fill PATH/TO/WALL
slstatus
exec dwm
```
